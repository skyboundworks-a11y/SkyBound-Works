# 🎬 Video Editing Portfolio Website

A high-end, premium portfolio website designed specifically to showcase video editing masterpieces and client testimonials. Built with a sleek dark aesthetic, modern glassmorphism features, and vibrant neon gradients.

---

## ✨ Key Features

* **Premium Glassmorphism:** Custom translucent cards (`glass-card`) with smooth backdrop-blurs and fine borders.
* **Dynamic Micro-interactions:** Fluid 500ms hover animations featuring scaling effects and glowing neon shadows.
* **Fully Responsive Grid:** A layout that adapts seamlessly from small mobile screens to 4K displays.
* **Crisp Iconography:** Vector-based SVG and Lucide icons that look razor-sharp on high-DPI displays.
